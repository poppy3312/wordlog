# Chrome Extension Icons

图标文件需要放在 `icons/` 目录下：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素
- `icon128.png` - 128x128 像素

## 设计建议

- 使用 Claude 温暖橙色 (#D97757)
- 简洁的书本图标 📚
- 或者字母 "W" + 书本元素
- 支持浅色/深色模式

## 生成工具

可以使用以下工具生成图标：
- Figma
- Canva
- Photoshop
- 在线图标生成器

## 临时方案

在开发阶段，可以使用在线图标库：
- https://www.flaticon.com/
- https://iconfinder.com/
- https://thenounproject.com/

搜索 "book" 或 "vocabulary" 关键词。
