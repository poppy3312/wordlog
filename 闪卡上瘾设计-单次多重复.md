# 在线单词闪卡：让用户「想再多看一组」的上瘾设计

> **定位**：闪卡模块 **专注闪卡学习**——只看、听、翻卡、完成一组、再来一组。不做测验、不判对错、不在此引导去其他模块，让用户心无旁骛地刷闪卡。  
> **目标**：单次「认词」上瘾，愿意「再来一组」「再刷一遍」。

---

## 一、闪卡模块：只做闪卡学习

- **专注一件事**：在本模块内，用户只做「看词图 → 翻卡 → 听/看词 → 下一张 → 完成一组 → 再来一组」。不穿插练习入口、不强调「去练一练」，避免打断闪卡心流。
- **边界**：测验、选答案、判对错、错题等放在**练习模块**；闪卡里只做「呈现 + 翻卡 + 进度 + 多重复」。
- 上瘾设计全部围绕**看完一张 / 看完一组**，让用户想一直留在闪卡里多刷几组。

---

## 二、核心心理学杠杆（为什么人会想「再来一次」）

| 机制 | 含义 | 在「认知向」闪卡里的用法 |
|------|------|---------------------------|
| **目标梯度效应** | 越接近目标，动力越强 | 每组 5 张 + 明显进度，第 4 张时「就差一张」的冲动 |
| **未完成张力（蔡格尼克效应）** | 没做完的事会在心里挂着 | 再完成一组就「满」，制造小缺口 |
| **可变奖励** | 奖励不可预测时，人更想继续 | 翻卡/看完一张时，有时普通、有时小彩蛋（动效/文案） |
| **即时反馈** | 每个动作都有立刻的反馈 | 翻卡动效、发音、进度格填充 |
| **完成感** | 做完一小段就有满足感 | 每看完 5 张 = 完成一组，有庆祝 |
| **心流** | 节奏紧凑、门槛低 | 单组短（5 张）、少等待，「再来一组」一点即开 |

---

## 三、产品设计清单（认知向 + 多重复）

### 1. 进度可视化 + 「差一点就完成」（必做）

- **每组 5 张**：5 个圆点或一条进度条，每**看完**一张就填满一格（不是「答对」才填）。
- **文案**：第 3 张「还剩 2 张」；第 4 张「最后 1 张！」。
- **完成瞬间**：整组看完后庆祝（动画 + 音效 + 「本组看完！」），然后**同一屏**出现「再来一组（5 张）」。
- **原理**：目标梯度 + 蔡格尼克效应，让人想「把这组看完」以及「再完成一组」。

### 2. 翻卡与认知节奏（必做）

- **先图后词**：正面以图为主（可选带词），点击翻到背面：词 + 发音（自动或点击播放）。
- **翻卡反馈**：翻转有轻动画 + 可选音效（翻页/弹出）；播放发音时若有波形或小图标反馈更好。
- **下一张**：用户点击「下一张」或滑动进入下一张，进度格随之填充。不设「对错」，只设「看过了」。
- **原理**：每个动作都有即时反馈，形成「翻卡 → 听/看 → 下一张」的稳定节奏。

### 3. 可变奖励（让「看完一张」有时不一样）（高性价比）

- **常规**：翻卡 → 正常动效 + 发音。
- **随机彩蛋**（如 15%～20% 概率）：翻到某张时触发——
  - 轻微特效（词卡闪光、pingu 探头等）；
  - 或一句随机轻文案（「这个词很常用哦」「pingu 也喜欢这个！」）。
- **原理**：可变奖励让人期待「下一张会不会有彩蛋」，愿意多刷几组。

### 4. 「看完一组」的庆祝（必做）

- **整组看完**：5 个进度格全满 → 短庆祝（动画 + 音效 + 「本组看完！」）。
- **可选**：本组 5 张都「翻过」且听过发音时，可触发「本组都听过啦」类轻文案（不涉及对错）。
- **原理**：完成感来自「看完/听过」，而不是「答对」，与认知定位一致。

### 5. 「再来一组」零摩擦（必做）

- **一组结束后**：同一屏内——
  - 本组小结（如「看完 5 个词」）；
  - 大按钮：「再来一组（5 张）」；
  - 次要：「换一组词」「今天先到这里」。（不在此放「去练一练」等入口，专注闪卡学习。）
- **默认焦点**：「再来一组」主按钮，一点即开下一组，无多余跳转。
- **原理**：减少决策成本，让「再多看一组」成为顺手操作，用户留在闪卡里持续刷。

### 6. 轻量悬念（下一组是什么）（可选）

- 在「再来一组」旁写「下一组：动物 / 食物……」或下一组的 1 个示例图/词。
- 或「下一组随机 5 个词」制造「开盲盒」感。
- **原理**：好奇心驱动「再开一组看看」。

### 7. 节奏不拖沓（可选）

- 用户可点击「下一张」或滑动；若希望自动推进，可设「播完发音后 X 秒自动下一张」（可关闭）。
- 单张停留时间由用户主导，避免长时间卡在同一张。
- **原理**：节奏清晰，易进入心流，自然想「再玩一组」。

### 8. 音效与动效（可选但加分）

- **翻卡、播发音、完成一组**：统一风格的短音效（可关闭）。
- 动效：翻卡、进度条填充、组完成庆祝，与 pingu/3D 风格一致。
- **原理**：多感官反馈强化「爽感」，更愿意重复。

---

## 四、单次认知流程示例（上瘾向）

1. 进入闪卡 → 看到「本组 5 张」+ 5 个空心圆点。
2. 第 1 张：正面看图（可先猜）→ 点击翻卡 → 背面词 + 发音 → 动效/可变彩蛋 → 第 1 格填满 → 点「下一张」。
3. 第 2～5 张：同上，进度逐步填满，偶尔出现轻彩蛋。
4. 第 5 张看完 → 5 格全满 → 「本组看完！」庆祝（动画 + 音效）。
5. **同一屏**：「看完 5 个词」+ 大按钮「再来一组（5 张）」+ 小字「换一组词 / 结束」。
6. 点「再来一组」→ 直接进入下一组 5 张，循环。

全程无「对错」、无跳转去练习，只有「看、听、下一张、完成一组、再来一组」——闪卡模块专注闪卡学习。

---

## 五、针对小学生可以加强的点

- **先图后词 + 发音**：符合你现有认知设计，强化音形义联结。
- **pingu 参与**：在翻卡彩蛋、组完成庆祝里用 pingu 表情/动作，增强情感连接。
- **时间可控**：单组 5 张约 1～2 分钟，「再玩两组」= 约 5 分钟，家长易接受。
- **专注闪卡**：组结束后不引导去练习模块，让用户留在闪卡里多刷几组；练习由用户从产品其他入口自行进入。

---

## 六、总结：优先做哪几件事（认知向）

| 优先级 | 做什么 | 目的 |
|--------|--------|------|
| **P0** | 每组 5 张 + 进度条/圆点（按「看完」填满）+ 组完成庆祝 + 同一屏「再来一组」大按钮 | 完成欲 + 零摩擦重复 |
| **P0** | 先图后词 + 翻卡动效 + 发音反馈；「下一张」明确，无对错 | 认知节奏清晰，每步有反馈 |
| **P1** | 翻卡/看完一张时的可变奖励（随机彩蛋/轻文案） | 期待感，想多刷几组 |
| **P1** | 组结束仅保留「再来一组 / 换一组 / 结束」，不引导离开闪卡 | 专注闪卡学习，心无旁骛多重复 |
| **P2** | 下一组预告/随机感、可选音效、可选自动进下一张 | 悬念 + 爽感 |

闪卡模块**专注闪卡学习**：看、听、翻卡、完成一组、再来一组。不在此做测验、不引导去练习，上瘾设计只围绕「多看、多听、多完成几组」，让用户愿意一直刷下去。
